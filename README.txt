Instrucciones para utilizar este repositorio:

Clonar url del repositorio (git clone url)
Crear un archivo o modificar archivo existente
Usar comando para agregar archivo (git add)
Realizar un commit (git commit -m "Descripcion"
Pushear los cambios al repositorio remoto (git push)
En caso de haber cambios en el repositorio remoto y querer adquirirlos en el local (git pull)
